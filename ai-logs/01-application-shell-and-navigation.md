# Application Shell and Navigation

## Primary Prompt

Build a mobile-first React application shell for a family pet companion app using Vite and TypeScript. Create a bottom-tab style navigation and separate pages for home, connect/feed, chat, profile, jar, and weekly echo. Keep the visual style warm, playful, and family-friendly. Use reusable components, keep state manageable, and structure the code so each page lives in its own file under `src/app/pages/`.

## Expected Output

- Main app layout and routing structure
- Shared visual language across pages
- Navigation between major sections
- Reusable UI building blocks for page composition

## Related Areas

- `src/app/pages/`
- `src/app/components/`
- `src/app/context/`
