# Profile and Family Management

## Primary Prompt

Create a `ProfilePage` where users can view the current account, switch among family members, add new family members, and remove existing ones. The page should work in both demo mode and real backend mode, fetch the current user and family list, and keep the interaction simple for a mobile family app. Use clean modal or panel interactions and handle loading and failure states gracefully.

## Expected Output

- Current user display
- Family member list management
- Add / delete member actions
- Account switching behavior

## Related Areas

- `src/app/pages/profile-page.tsx`
- `backend/routes/family-members.js`
- `backend/routes/me.js`
