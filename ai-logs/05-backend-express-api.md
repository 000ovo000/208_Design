# Backend Express API

## Primary Prompt

Build an Express backend for a family pet companion application. Expose REST-style routes for current user lookup, family members, moods, posts, uploads, item storage, daily drop, weekly echo, care messages, and pet messages. Connect the server to MySQL, return JSON responses, and keep the API straightforward for a Vite frontend running locally. Serve uploaded files from a static uploads directory and keep the route modules separated by feature.

## Expected Output

- Express server entry point
- Feature-based route modules
- MySQL connection layer
- Upload handling and static file serving

- `backend/server.js`
- `backend/db.js`
